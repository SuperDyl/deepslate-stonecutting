## Deepslate Stonecutting

> [!IMPORTANT]
> This data pack no longer receives updates.
> It's features are part of the game as of 1.26.1 (26.1 Snapshot 8)
> The current version should work for all versions of the game before the specified version.

This datapack adds stonecutting recipes for deepslate to match cobbled deepslate.
Every stonecutter recipe for cobbled deepslate is added for deepslate.
